# 港股 + 美股 日频趋势突破策略说明

本策略为**日频、纯技术面、多头趋势突破**，**不含基本面**。下文为**默认参数**下的逻辑与计算方法。

---

## 一、策略概要

在每个交易日，在允许**新开仓**的前提下，从**候选池**中选出满足**突破 + 趋势 + 量能 + 流动性**条件的标的，按规则建仓；持仓按**止损、移动止盈、唐奇安下轨、可选 MA60 亏损离场**依次检查是否卖出。仓位可随**波动率目标**缩放。

---

## 二、选股与候选池

1. **推荐默认（恒指 ∪ 恒生科技）**  
   - **恒指成分**：以恒生指数公司公布的成分名单为准（需自行维护成分表 CSV）。  
   - **恒生科技成分**：同上。  
   - 两者取**并集**去重；恒指与恒生科技重叠较多，**合并后约 90± 只量级**。

2. **自选列表**：仅交易你指定的代码列表（`UNIVERSE_MODE=csv` 与股票池 CSV）。

**每日可交易池**：在候选池内，再按**截至昨收**的**价格区间**与**近若干日平均成交额区间**（港股、美股分别设定上下界）过滤；只有同时满足流动性条件的标的才会进入当日买入池。

---

## 三、时间与信号计算（避免未来函数）

- 回测日历日 \(T\) 上，任何决策仅使用**不晚于 \(T-1\)** 的日线。  
- **突破**：收盘价高于「前一日为止的 \(N\) 日最高价」，即前一日滚动最高价在**再往前一日**的基础上比较，**当日最高价不参与**突破判定。  
- **趋势**：收盘价高于 \(P\) 日移动平均（默认 \(P=50\)）。  
- **量能**：当日成交量相对成交量均线的比值 ≥ 阈值（默认 1.2）。  
- **流动性**：近若干日平均成交额落在设定区间内。  
- **成交价假设**：信号按 \(T-1\) 日收盘价形成，成交价用该收盘价，等价于「按昨收成交」的简化；**未模拟**盘中滑点、跳空与冲击成本，实盘通常更差。

---

## 四、大盘环境过滤（默认开启）

仅在 **恒生指数** 与 **标普 500 代表性 ETF（美股）** 的收盘价**均**高于各自 **\(M\) 日移动平均**（默认 \(M=200\)）时，允许**新开仓**；已有持仓仍可按出场规则卖出。  
若改为「**任一**指数在均线上即可」，则条件更宽松，需慎用。

---

## 五、波动率目标仓位（默认开启）

用标的近若干日对数收益估计**年化波动率**，将单笔目标名义仓位按「目标年化波动 ÷ 实现波动」缩放，并夹在**下限**与**上限**之间。高波动少买、低波动略多买，使各标的贡献相对均衡。目标年化波动设为 **0** 时等价于关闭该缩放。

---

## 六、出场规则（按顺序）

1. **固定止损**：相对买入价亏损超过设定比例则清仓。  
2. **移动止盈**（默认开启）：浮盈达到设定比例后，若收盘价自持仓以来**最高价**（由日线收盘价递推的峰值）回撤 ≥ 设定比例，则清仓。  
3. **唐奇安下轨**：收盘价跌破「前一日为止的 \(D\) 日最低价」（默认 \(D=20\)），清仓。  
4. **亏损 + MA60**：若开启，收盘价跌破 60 日均线且当前为亏损，则清仓。

---

## 七、仓位与交易单位

- 最多同时持有**最多持仓数**只；单笔目标为总权益的**仓位比例**，再乘以波动率缩放。  
- 港股按**整手 100 股**近似；美股按**整股**。  
- 单边费率（默认 0）可按买卖各一次自行理解口径。

---

## 八、业绩基准（与策略对比用）

报告中「基准」为 **恒指与美股 SPY 收盘价等权、各自归一化到起点 100 后再取平均**得到的组合曲线，用于粗略对比跨市场买入持有风格。

---

## 九、默认参数一览

| 含义 | 默认值 |
|------|--------|
| 训练期（仅 `train_params.py` 使用；拟合参数） | 2020-01-01 ～ 2022-12-31 |
| 主回测 / 样本外验证区间（`backtest.py` 的 `BACKTEST_*`） | 2024-01-01 ～ 2025-12-31 |
| 突破回望天数 \(N\) | 44（与默认训练选优一致，可改） |
| 趋势均线周期 \(P\) | 50 |
| 成交量均线周期 | 20 |
| 量比阈值 | 1.2 |
| 唐奇安出场天数 \(D\) | 20 |
| 固定止损比例 | 8%（与默认训练选优一致，可改） |
| 大盘过滤 | 默认关闭（训练选优结果）；若开启，则恒指与美股 SPY 均在 200 日均线上才允许新开仓 |
| 波动率目标（年化） | 15%（设为 0 关闭） |
| 实现波动估计窗口 | 20 交易日 |
| 波动缩放上下限 | 0.35～2.5 倍 |
| 移动止盈：激活浮盈 / 峰值回撤 | 12% / 9%（任一为 0 则关闭） |
| 最多持仓数 | 10 |
| 单笔目标占权益比例 | 20% |
| MA60 亏损离场 | 开启 |
| 港股近若干日平均成交额 | 约 500 万～500 亿（港元口径） |
| 美股近若干日平均成交额 | 约 200 万～500 亿（美元口径） |

## 十、局限与免责声明

- 历史回测不预示未来；股票池过小或时段过短会**严重扭曲**夏普、回撤与年度胜率。  
- 若首段数据始于某年中，该「首年」可能是**非完整自然年**。  
- 当前默认参数为**风险与收益权衡**下的一组设定，**不保证**在任何样本外区间或实盘中仍为最优。  
- **样本划分（可选做法）**：训练窗口在 `train_params.py`；主回测区间只在 `backtest.py` 配置，二者分离。
